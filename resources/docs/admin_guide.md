# Admin Guide (Outline)

## Overview
- Purpose of the platform and roles (platform admin vs org admin)
- Privacy statement reminder

## Onboarding an Organization
- Add org record (Admin > Orgs)
- Set org status active/disabled
- Configure org code / portal links

## Users & Roles
- Create users, assign roles (org_admin, reviewer, security_lead, platform_admin)
- Activate/deactivate accounts
- Resetting passwords

## Alert Recipients
- Configure alert contacts (email/SMS) per org
- Departments and when they’re used (student vs employee)
- Urgent alerts vs regular notifications

## Portals (General / Student / HR)
- Portal URLs and when they’re enabled/disabled
- Branded disabled page behavior
- Force portal types per org

## Categories & Subcategories
- Manage categories (Admin > Categories)
- Type mapping (student/employee/both)
- HR category presets

## Review Queue & Permissions
- Org scoping for non-platform admins
- Trash restore

## Notifications & SLA
- Urgent alerts (email/SMS), reporter follow-up alerts
- First response timestamp/notification to org admins (config toggle)

## Analytics
- Filters (portal, org, date range)
- Metrics available (counts, urgent %, avg first response, type/severity/category, org breakdown)
- CSV export

## Data Retention & Storage
- Where files are stored (public disk by default)
- Optional storage monitoring (planned)

## Security & Compliance
- Authentication, reset, 2FA (if enabled)
- Audit trail (Activity History uses AuditLog)
